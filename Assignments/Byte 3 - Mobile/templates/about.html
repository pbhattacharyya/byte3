Accountability: About This Data Set

I used the AWARE framework to use my smartphone's sensors to collect data about my location, movement, and activities. I collected the following data: accelerometer, location, and activity recognition.

Where did this data come from?

This dataset comes from data I collected on my phone using the AWARE framework. More info at http://www.awareframework.com/

While cleaning the location dataset, I plotted all the datapoints to make sure that they all mapped to the Pittsburgh area. I also converted the unix timestamp to human time to verify authenticity.

While preparing the acitivity recognition dataset, I plotted all the datapoints on a time axis and visualized the data using color to make sure that all the data was correct. That's how I noticed that a few datapoints from Feb 28 were wrong - the dataset implied that I was driving at 3 AM, when in reality I was fast asleep.

For the accelerometer dataset, I queried the dataset using SQL to extract the rows for which the square root of the sum of the squares of doubles_value_0, doubles_value_1, and doubles_value_2 were equal to or close to 0 (a value of 0 implies freefall, i.e., a phone being dropped from a height).

Who has access to it?

The data in these visualizations is publicly accessible but owned by me. The rest of the data (I collected data over 7 days) is private.
